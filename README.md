# Machine Learning Programs

This repository contains **Machine Learning programs implemented using Python**.
The main objective of this repository is to practice and understand different **Machine Learning algorithms such as Linear Regression, Logistic Regression, and Polynomial Regression**, along with model evaluation techniques.

The programs demonstrate how to train models, evaluate their performance, and visualize results.

---

## 📁 Repository Structure

The repository contains the following files:

* **SimpleLR.py** – Implementation of Simple Linear Regression
* **multiple LR.py** – Multiple Linear Regression model
* **linear reggerssion.py** – Basic Linear Regression example
* **Polynomial Regression.py** – Polynomial Regression model implementation
* **Logistic Regression with Confusion Matrix.py** – Logistic Regression with model evaluation
* **Logistic Regression with Confusion Matrix2.py** – Logistic Regression with extended evaluation
* **Logistic Regression with New Candidate Prediction.py** – Predicting a new data point using Logistic Regression
* **Confusion Matrix & Heatmap.py** – Visualization of confusion matrix using heatmap
* **checkinglinearity.py** – Checking linearity in regression data
* **lr2.py** – Additional regression example
* **position_salaries.csv** – Dataset used for regression model training
* **stock.csv** – Dataset used for regression analysis

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Gouthamvaishnav11/Machine-Learning-programs.git
```

Navigate to the project folder:

```
cd Machine-Learning-programs
```

Install required libraries:

```
pip install numpy pandas matplotlib seaborn scikit-learn
```

---

## ▶️ Running the Programs

Run any Python file using:

```
python filename.py
```

Example:

```
python SimpleLR.py
```
